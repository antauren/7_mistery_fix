# Quadratic Equations Solver

Программа решения квадратных уравнений.

# How to Use

```bash
python quadratic_equation.py 1 2 -3
#Уравнение имеет два решения:  -3.0 1.0
```
# How to Help

```bash
python quadratic_equation.py --help
```

# How to Launch Tests

```bash
python tests.py
```
